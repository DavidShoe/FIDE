RUN UUT

Description

Allows the interface pod microprocessor to execute the program code stored in the UUT.
  • If an address is specified, the UUT begins executing the code at the address indicated.

  • If no address is specified but a RUN UUT setup parameter is present, the addre from the setup statement is supplied.

  • If no RUN UUT statement appeared in the setup section, but a pod data file was include at the beginning of the source file, then the value for UUTADR will be supplied.

  • If a pod data file was not included at the beginning of the source file, the default address i 0000.

Example

run UUT
run UUT @ 1000
