POD

Description

Identifies the pod to be used when executing the 9010A program in the source file. The POD statement allows the 9010A to use the data in the pod data file to configure its setup parameters to match the specified pod.

  • At this time, the following pods are valid:
      1802, 6502, 6800, 68000, 6802, 6809, 6809E, 8041, '35/48, '39/49, '40.50, 8080, 8085, 8086, 8086MAX, 8088, 8088MAX, 9900, Z80


Example

pod 8080
pod '39/49
pod Z80
