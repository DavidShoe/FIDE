IF

Description

Creates conditional branch steps.
  • Symbolic label names can be used in the statement.

  • Within a single program, symbolic lables cannot be mixed with hexadecimal label number (0-9, A-F).

  • More than on IF step may redirect program execution to the same label.

  • The label to which program execution is redirected may appear anywhere in the program.

Example

if REG3 and 7F > REG 4 goto 1
See also: label, goto.
