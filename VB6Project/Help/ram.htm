RAM TEST

Description

RAM SHORT quickly identifies common RAM failures such as address decoding errors or bits that are not read-writeable. RAM LONG performs the same tests as RAM SHORT and in addition, performs a pattern-sensitivity test for location "soft" RAM errors.
  • If no addr block is specified, then the 9010A performs RAM test over all blocks of memory specified as RAM under the Address Space Information.

Example

RAM short @ 1000 - 3FFF
RAM long
