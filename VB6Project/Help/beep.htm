BEEP

Description

Setup parameter.

Allows the programmer to control whether or not the 9010A should beep on ERR TRANSISTIONS.
  • YES enables the audible beep that sounds whenever an error is detected and reports. The beep also sounds whenever the error is remved.

  • The 9010A's default value is YES.

Example

beep on err transition NO
beep on err transition YES

See also: trap, exercise, enable.
